## 🎨 核心特效组件

### 1. 极光背景（Aurora）
- **HTML 结构**：
```html
<div class="aurora-layer" aria-hidden="true">
  <div class="aurora-band aurora-band--blue"></div>
  <div class="aurora-band aurora-band--purple"></div>
  <div class="aurora-band aurora-band--green"></div>
  <div class="aurora-fade"></div>
</div>
```

- **CSS 关键参数**：
  - `aurora-band`: 基础样式，模糊和混合模式
  - `aurora-band--blue`（蓝）：opacity 0.58
  - `aurora-band--purple`（紫）：opacity 0.82  
  - `aurora-band--green`（绿）：opacity 0.76
  - 动画持续时间：19s/23s/27s（可调整）

### 2. 反重力粒子效果（Antigravity Particles）
- **Canvas 元素**：
```html
<canvas id="antigravityCanvas" class="antigravity-layer" aria-hidden="true"></canvas>
```

- **配置参数**（在脚本中调整）：
```javascript
const config = {
    count: 1200,           // 粒子数量
    magnetRadius: 140,     // 吸引半径
    ringRadius: 112,       // 环形半径
    waveSpeed: 0.018,      // 波形速度
    waveAmplitude: 10,     // 波形振幅
    lerpSpeed: 0.08,       // 平滑速度
    particleSize: 5.5,     // 粒子大小
    pulseSpeed: 0.08,      // 脉冲速度
    autoAnimate: true      // 自动动画
};

const colors = ["rgba(0,0,0,1)", "rgba(15,15,15,1)", "rgba(30,30,30,1)"];  // 粒子颜色
```

### 3. 文字光泽效果（Shiny Text）
- **HTML**：
```html
<span class="shiny-text" 
  data-shiny-speed="2" 
  data-shiny-direction="left" 
  data-shiny-delay="0.2" 
  data-shiny-pause-on-hover="true">文字内容</span>
```

- **属性说明**：
  - `data-shiny-speed`: 动画速度（默认 2）
  - `data-shiny-direction`: 扫光方向（left/right，默认 left）
  - `data-shiny-delay`: 延迟时间（秒）
  - `data-shiny-pause-on-hover`: 悬停暂停（true/false）

## 🔧 集成步骤

### Step 1: 复制依赖
```html
<!-- 在 <head> 中添加 -->
<script src="https://cdn.tailwindcss.com?plugins=forms,container-queries"></script>
<link href="https://fonts.googleapis.com/css2?family=Manrope:wght@400;700;800&family=Inter:wght@400;500;600&family=Space+Grotesk:wght@400;500;700&display=swap" rel="stylesheet"/>
<link href="https://fonts.googleapis.com/css2?family=Material+Symbols+Outlined:wght,FILL@100..700,0..1&display=swap" rel="stylesheet"/>
```

### Step 2: 配置 Tailwind 主题
```javascript
<script id="tailwind-config">
tailwind.config = {
  darkMode: "class",
  theme: {
    extend: {
      colors: {
        // ... (参考完整代码中的色值配置)
      },
      fontFamily: {
        "headline": ["Manrope"],
        "body": ["Inter"],
        "label": ["Space Grotesk"]
      }
    }
  }
}
</script>
```

### Step 3: 添加 CSS 样式
- 复制 `<style>` 部分中的所有 CSS
- 特别关注：
  - `.aurora-*` 系列样式
  - `.antigravity-layer` 
  - `.shiny-text` 和相关 CSS 变量
  - 所有 `@keyframes` 动画

### Step 4: 复制 HTML 结构
```html
<!-- 在 <body> 开始位置添加背景层 -->
<div class="aurora-layer" aria-hidden="true">...</div>
<canvas id="antigravityCanvas" class="antigravity-layer" aria-hidden="true"></canvas>

<!-- 在内容区域添加 Hero section 和 ShinyText -->
<h1 class="font-headline font-extrabold text-[48px] md:text-[80px]">
  <span class="block shiny-text" data-shiny-speed="2" data-shiny-direction="left" data-shiny-delay="0.2">您的愿景</span>
  <span class="shiny-text" data-shiny-speed="2.3" data-shiny-direction="left" data-shiny-delay="0.35">我们的数字现实</span>
</h1>
```

### Step 5: 添加 JavaScript
- 复制三个 IIFE 函数（立即执行函数表达式）：
  1. **按钮逻辑**：`startAnalysisBtn` 点击处理
  2. **Shiny Text 动画**：`shinyNodes` 扫光效果
  3. **Antigravity 粒子系统**：Canvas 粒子渲染和交互

### Step 6: 配置资源路径
- 修改 `body` CSS 中的 `background-image: url()` 指向你的图片
- 修改按钮跳转链接 `window.location.href` 指向你的页面路径

## ⚙️ 常用调参

### 调整极光强度
编辑 `.aurora-band` 和各色系的 `opacity` 值：
```css
.aurora-band {
    opacity: 0.72;  /* 增加此值使极光更明显 */
}
```

### 调整粒子数量和大小
```javascript
const config = {
    count: 1200,        // 减少此值降低性能消耗
    particleSize: 5.5,  // 增加使粒子更大
};
```

### 调整粒子颜色
```javascript
const colors = ["rgba(R,G,B,A)", ...];  // 修改 RGB 和透明度 A
```

### 禁用某个特效
```css
/* 禁用极光 */
.aurora-layer {
    display: none;
}

/* 禁用粒子 */
.antigravity-layer {
    display: none;
}
```

## 🎯 性能优化建议

1. **粒子优化**：
   - 生产环境建议 `count: 600-800`
   - 监控 FPS，必要时降低 `particleSize`

2. **极光优化**：
   - 移动设备可降低 `opacity`
   - 考虑使用 `transform: translateZ(0)` 启用 GPU 加速

3. **图片优化**：
   - 背景图片压缩至 200-300KB
   - 考虑使用 WebP 格式

## 📝 注意事项

1. 确保背景图片路径正确，相对于 HTML 文件位置
2. Tailwind CDN 需要网络连接，生产环境建议自托管
3. Material Icons 依赖网络字体加载
4. Canvas 粒子在低端设备可能性能不佳，考虑检测 CPU 核心数
5. 某些浏览器可能需要 `-webkit-` 前缀支持
