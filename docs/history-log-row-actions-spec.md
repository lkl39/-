# 历史日志行操作规格

## 范围
- 页面：`public/inner-pages/历史日志存档/code.html`

## 规则
- 历史日志表格行内操作仅保留当前稳定能力。
- 已完成记录显示：
  - `查看报告`
  - `删除`
- 分析中记录显示：
  - `处理中`
  - `删除`

## 禁止项
- 不再显示行内 `下载` 图标按钮。
- 顶部批量导出入口不在本次调整范围内。

## 删除行为
- 点击行内 `删除` 后，必须先二次确认。
- 确认后调用 `/api/inner-data` 的 `history-delete` 动作。
- 删除成功后，当前表格、筛选结果和分页状态立即刷新。
- 删除失败时，仅提示错误，不刷新整页。
