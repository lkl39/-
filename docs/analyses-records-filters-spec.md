# 分析记录筛选规格

## 范围

- 页面：`public/inner-pages/分析记录/code.html`
- 数据来源：`GET /api/inner-data?view=analyses`

## 目标

- 让分析记录页的文件名搜索可用。
- 同步接通分析状态、风险等级筛选与分页刷新。
- 不改后端字段结构，不新增高成本接口。

## 交互规则

- 文件名搜索：
  - 输入后短暂停顿自动筛选
  - 按 `Enter` 立即筛选
- 分析状态变更后立即筛选
- 风险等级变更后立即筛选
- 时间范围输入本次仅保留展示，不做真实筛选

## 分页规则

- 每页 8 条
- 筛选后回到第 1 页
- 分页文案与页码按钮随筛选结果同步更新
